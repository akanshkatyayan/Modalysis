## Setup: Local Deployment
```
git clone https://github.com/antra0497/Modalysis.git
cd Modalysis
pip install -r requirements.txt
```
Download pretrained Bert Model:
```
cd user
python model_download.py
```
Run the Flask server
```
python wsgi.py
```
